# Readest 交互功能移植需求文档

## 1. 项目概述

**目标**：将 Readest 阅读器的优秀交互体验移植到 TactiReader，提升 PDF 阅读的便利性和现代化体验。

**当前 TactiReader 状态**：
- 支持 PDF 阅读、批注、书签、主题切换
- 支持多文档标签页、快捷键操作
- 基于 PyQt5 的桌面应用

---

## 2. Readest 核心交互功能分析

### 2.1 阅读模式

| 功能 | Readest 实现 | TactiReader 现状 | 移植优先级 |

| 滚动模式 | ✅ 支持上下滑动 | ❌ 不支持 | **高** |

| 功能 | Readest 实现 | TactiReader 现状 | 移植优先级 |
|------|-------------|-----------------|-----------|

| 添加笔记 | ✅ 关联文本段落 | ❌ 仅图形批注 | **高** |
| 笔记侧边栏 | ✅ 集中管理 | ❌ 不支持 | **高** |
| 笔记导出 | ✅ 支持导出 | ❌ 不支持 | 中 |

### 2.3 辅助功能

| 功能 | Readest 实现 | TactiReader 现状 | 移植优先级 |

## 3. 高优先级移植功能详细需求

### 3.1 滚动阅读模式

**需求描述**：
- 支持垂直滚动阅读，类似网页浏览体验
- 可通过快捷键或菜单切换"分页模式"和"滚动模式"
- 滚动模式下保持批注和书签功能正常

**技术方案**：
```python
# 新增滚动模式状态
self.scroll_mode = False  # False=分页, True=滚动

# 新增快捷键
# Ctrl+R 或菜单项切换模式

# 实现要点
# 1. 在滚动模式下，将所有页面垂直排列
# 2. 使用 QScrollArea 包裹所有页面
# 3. 保持批注坐标系统兼容
```

**涉及文件**：
- `tactireader.py` - 主窗口逻辑
- 新增 `scroll_view.py` - 滚动视图组件

---

### 3.2 文本笔记系统

**需求描述**：
- 选中文本后可添加文字笔记
- 笔记与文本段落关联，高亮显示
- 侧边栏集中查看所有笔记

**技术方案**：
```python
# 笔记数据结构
note = {
    "page": 1,
    "text_selection": {
        "start_offset": 100,
        "end_offset": 200,
        "text": "选中的文本内容"
    },
    "note_content": "用户添加的笔记",
    "created_at": "2026-06-19 21:00:00",
    "color": "#FFD700"
}

# 实现要点
# 1. PDF 文本提取：使用 PyMuPDF 的 page.get_text()
# 2. 文本选择：实现文本选择交互
# 3. 笔记存储：扩展现有 annotations 结构
# 4. 侧边栏：新增笔记面板，类似书签面板
```

**涉及文件**：
- `tactireader.py` - 笔记管理逻辑
- 新增 `note_panel.py` - 笔记面板组件
- `config.py` - 扩展数据结构

---

### 3.3 笔记侧边栏

**需求描述**：
- 左侧或右侧显示所有笔记列表
- 点击笔记跳转到对应页面
- 支持搜索、编辑、删除笔记

**技术方案**：
```python
# 笔记面板 UI
class NotePanel(QWidget):
    - 笔记列表 (QListWidget)
    - 搜索框 (QLineEdit)
    - 编辑区 (QTextEdit)
    - 操作按钮 (编辑/删除/跳转)
```

**涉及文件**：
- 新增 `note_panel.py`
- `tactireader.py` - 集成面板

### 4.2 字体自定义

**需求描述**：

- 支持字体大小调节

**技术方案**：
```python

class FontSettingsDialog(QDialog):
    - 字体选择 (QFontComboBox)
    - 字号调节 (QSpinBox)
    - 预览区 (QLabel)
```

### 4.3 文本转语音 (TTS)

**需求描述**：
- 朗读选中文本或当前页面
- 支持语速、音量调节
- 支持暂停/继续

**技术方案**：
```python
# 使用 pyttsx3 或 Windows SAPI
import pyttsx3

class TTSManager:
    def __init__(self):
        self.engine = pyttsx3.init()
    
    def speak(self, text):
        self.engine.say(text)
        self.engine.runAndWait()
```
