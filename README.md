# TactiReader（战术阅读器）完整教程总结
## 一、软件定位与开发背景
### 1. 核心痛点
常规PDF阅读器（WPS、Adobe）默认线性阅读逻辑，不适合专业教材：扫描PDF无超链接、跨章节频繁跳转、页码错位、多窗口对照杂乱，来回翻找打断阅读思路；纸质书可卡页、双页对照、快速回溯，电子工具无法复刻该体验。

### 2. 产品定位
个人开发的专业非线性PDF阅读器，被称为**认知外骨骼**，核心目标：复刻纸质书自由卡页、多页对照、瞬时跳转的阅读逻辑，适配高数、单片机、工程手册等交叉引用密集的长篇专业文档。
开发周期3周，Python开发，纯本地离线工具。

### 3. 适用场景
- 数学、矩阵论、工程、嵌入式、单片机长篇教材/技术手册
- 扫描版PDF（无内置超链接、目录残缺、页码与正文错位）
- 需要频繁跨章节对照公式、图表、附录、定义的深度阅读

## 二、核心特色功能与操作教程
### （一）双窗格对照（核心功能）
1. 切换：`Z` 单/双窗格切换
2. 联动/独立模式：`C` 一键切换
   - 默认联动：左窗格跟随右窗格前一页，连贯阅读
   - 独立模式：左右页面互不干扰，可一边放图表、一边读正文
3. 焦点区分：点击窗格切换操作对象，激活窗格边框变红，避免误操作
4. 独立视图：左右窗格可单独缩放、旋转（`~`旋转90°），拖动中间分隔条调整两栏宽度

### （二）Q-P十键盲操书签（纸质书签平替）
键盘上排 `Q W E R T Y U I O P` 10个按键专属书签，全程不用鼠标侧边栏
- `Ctrl+字母`：新建书签，可自定义命名，也可直接回车留空
- 单按对应字母：一键跳转到该书签页
- `Alt+字母`：删除对应书签
优势：盲操、不打断阅读心流，速度比鼠标书签快10倍

### （三）双路线页面回溯，解决迷路问题
1. Home主线锚点
   - 空格：一键跳回阅读最远主线页
   - `Ctrl+空格`：手动重置主线起点（如从附录开始读可重置）
2. 单步回退 `Ctrl+A`：仅返回上一次跨页跳转前位置，左右窗格独立记录

### （四）快慢双轨翻页，高效浏览
1. 精细单页：`A`上一页 / `D`下一页
2. 批量快翻：数字1-9设置翻页倍率，`←/→`一次翻对应页数，快速掠过无关章节

### （五）逻辑页码校正（解决PDF封面占位页码错乱）
1. `Ctrl+G`：标定当前PDF物理页对应的真实正文页码
2. `G`：跳转校正后的逻辑页码，书签、搜索统一使用校正页码
3. `Ctrl+Shift+G`：清除页码偏移校准

### （六）视图与基础阅读功能
1. 视图切换
   - `N`：显示/隐藏书签侧边栏
   - `F11`：全屏阅读
2. 文字与批注
   - `J`：开启文字选取复制（仅文字PDF）
   - `B`画笔、`H`矩形高亮、`V`文本批注；`Ctrl+Z`撤销；`Ctrl+Shift+C`清空本页批注
3. 搜索：`F`全文搜索，匹配内容高亮，回车切换下一条结果
4. 目录：自动解析PDF内置目录，点击直接跳转

### （七）状态持久化与多文件管理
1. 自动保存：关闭软件自动留存窗口布局、缩放、书签、批注、页码校准；再次打开自动恢复阅读环境
2. 多标签页（2.2版本新增）：同时打开多份PDF切换阅读
3. 专属笔记格式 `.tactinote`：打包PDF+全部阅读标注，可迁移完整阅读环境

### （八）键鼠分工设计
- 键盘：跳转、书签、翻页、视图切换等高频离散操作，快捷键集中左手区，无需换手
- 鼠标：滚轮缩放页面、拖动分隔条、拖拽页面平移，负责精细视觉调整

## 三、软件基础优势
1. 纯净离线：无联网、无广告、无会员、永久免费，不读取隐私
2. 轻量低占用：安装包仅60MB，单exe一键安装，无需额外依赖；千页PDF加载流畅不卡顿
3. 全平台支持：Windows、Mac、Linux；中英双语界面、完整帮助文档，菜单可查全部快捷键
4. 开源可二次开发：源代码托管Gitee，支持用户提交反馈、PR优化

## 四、安装渠道
1. Windows 直装包：Gitee发布页下载exe，一键安装
2. Linux：解压tar.gz，直接运行可执行文件
3. Mac：zip压缩包，右键打开绕过安全拦截（版本更新略滞后）
4. Microsoft Store：搜索TactiReader（存在沙盒阉割，不推荐主力使用）
5. 源码地址：Gitee开源仓库，可自行编译

## 五、整体设计理念
1. 模拟纸质书**阅读逻辑**，而非纸张外观；核心是自由跳转、对照、标记
2. 所有操作语义唯一、无歧义，减少用户配置成本，不打断思考流程
3. 区别于通用阅读器，定位**专业深度阅读工具**，主打非线性查阅需求
