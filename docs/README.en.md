# TactiReader

TactiReader is a keyboard-first PDF reader for long technical documents, standards, manuals, and textbooks with dense cross-references.

## Highlights

- Dual-pane reading for side-by-side comparison
- Instant bookmarks on `Q` to `P`
- Fast page jumps with multipliers and calibrated logical page numbers
- Search, annotations, notebook export, and multi-tab workflows

## Run

```bash
pip install PyMuPDF PyQt5 markdown
python tactireader.py
```

Open a file directly:

```bash
python tactireader.py your_file.pdf
python tactireader.py your_notebook.tactinote
```
