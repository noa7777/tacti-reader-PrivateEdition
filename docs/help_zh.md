# TactiReader 深度解码：为扫描版 PDF 重塑"纸质级"非线性读写体验

## 一、 破局：它解决了什么根本痛点？

传统 PDF 阅读器（如 Adobe Acrobat）是"线性卷轴"逻辑，适合小说，却极不适合高交叉引用密度的工程技术手册、数学教材或扫描版古籍。这类书籍的阅读本质是非线性跳跃（从正文跳到附录图表，再跳回公式推导）。

TactiReader 的哲学：放弃"模拟书本翻页"的表象，直击"纸质书随心所欲"的灵魂——用"双窗格空间并置"替代"来回翻页的时间损耗"，用"键盘盲操书签"替代"鼠标菜单的视线转移"。


## 二、 核心双引擎：空间与记忆的数字化映射

### 1. 空间引擎：双窗格对照系统（视觉工作区）

这是 TactiReader 最硬核的底层交互，彻底改变了"单一视口"的局限：

* 默认联动（线性流）：左右窗格默认绑定，左侧自动跟随右侧，保证常规通读的连续性。

* 独立模式（对照流）：按下 C 键解锁联动。此时左右窗格彻底解耦，各自独立翻页。典型场景：左窗格锁定在第 300 页的复杂电路图，右窗格翻到第 50 页对应的文字说明，实现"图-文"零延时对照。

* 焦点机制（操作锚点）：单击任一窗格使其边框变红即为焦点窗格。切记：所有翻页（A/D）、跳转（G）、书签（Q~P）等操作，永远只作用在当前红色边框的焦点窗格上。这避免了双屏操作时的"指哪打哪"混淆。

### 2. 记忆引擎：十键盲操书签系统（脑内索引）

针对扫描版 PDF 缺乏超链接的硬伤，TactiReader 将键盘上排的 Q W E R T Y U I O P（10个键）直接映射为 10 个物理"脑内标签卡"：

* 打标签（写入）：Ctrl + Q（Q 代表上述任意键）→ 当前焦点页被永久记忆。

* 调标签（读取）：直接按下 Q（无需 Ctrl）→ 瞬间穿越回该页。

* 撕标签（删除）：Alt + Q → 清除该槽位记忆。

设计精髓：这组快捷键让您的左手始终稳定在键盘左半区（Ctrl/Alt）和上排字母区，右手控制鼠标或方向键，眼睛全程无需离开屏幕寻找菜单栏。一旦形成肌肉记忆，10 个关键锚点（如：Q=目录、W=核心定义、E=关键图表……）将比实体书签条更高效。


## 三、 场景化实战工作流（如何组合使用）

场景：阅读一本 800 页的扫描版高等数学教材

1. 开局标定：按 Ctrl + G 标定物理页码偏移（解决扫描版页码与 PDF 逻辑页码不一致的问题），然后按 G 输入真实页码直达正文。

2. 锚点布局：找到核心定理页，按 Ctrl + W 存为书签 W；找到附录积分表，按 Ctrl + E 存为书签 E。

3. 对照阅读：按 C 进入独立模式，左窗格翻到附录（按 E 跳转），右窗格继续通读正文（按 A/D 翻页）。突然遇到不懂的引理，按 W 瞬间将左窗格切回核心定理页对照，看完后再按 E 把左窗格切回附录。

4. 深度批注：遇到重点，按 B（画笔）、H（高亮）或 V（文字批注）随手标注，按数字键 1~9 快速切换 9 种颜色（批注模式下）；不满意按 Ctrl + Z 撤销。

5. 全局回溯：按 Space 自动跳转到本次阅读过的最大页码（主线进度），按 Ctrl + Space 手动将当前页设为主线程里程碑。


## 四、 终极快捷键逻辑树（一张图理清所有按键）

为便于记忆，将快捷键按逻辑域重新归类：

### 【导航域】
| 按键 | 功能说明 |
|------|----------|
| A / D | 上一页 / 下一页（最常用） |
| Ctrl + A | 单步回退（撤销上一次跳转，类似浏览器后退） |
| ← / → | 快退 / 快进 N 页（N 由数字键 1~9 设定） |
| G | 跳转指定页码 |
| Ctrl + G | 校准物理页码偏移 |
| Space | 跳转至主线页（阅读过的最大页码） |
| Ctrl + Space | 将当前页设为主线程 |

### 【书签域】
| 按键 | 功能说明 |
|------|----------|
| Q~P（直接按） | 跳转至对应 10 个槽位的书签 |
| Ctrl + Q~P | 设置当前页到对应槽位 |
| Alt + Q~P | 清除对应槽位书签 |

### 【视口域】
| 按键 | 功能说明 |
|------|----------|
| C | 切换左右窗格 联动/独立（核心开关） |
| Z | 单页 / 双页 显示模式切换 |
| X | 重置当前焦点窗格的缩放与位移（看乱了按一下恢复） |
| ~（波浪键） | 顺时针旋转 90°（专治扫描版歪斜） |
| F11 | 全屏沉浸模式 |

### 【批注域】
| 按键 | 功能说明 |
|------|----------|
| B / H / V | 进入画笔 / 高亮 / 文字批注模式（ESC 退出） |
| 1~9（批注模式下） | 切换当前工具的 9 种颜色（默认为纯红，文字批注默认 25pt） |
| Ctrl + Z | 撤销上一步批注（支持多步） |
| J | 文字选取模式（复制文本用） |

### 【全局辅助域】
| 按键 | 功能说明 |
|------|----------|
| F | 全文搜索（扫描版需 OCR 支持） |
| L | 弹出目录对话框（顶部显示 Q~P 书签栏，下方自动展开 TOC 树） |
| N | 显示/隐藏侧边目录面板 |
| S | 立即保存状态（书签、批注、窗口位置），防止意外丢失 |


## 五、 总结：回归纸质书的掌控感

TactiReader 并非简单的"双开 PDF"，它通过空间分离（C键）解决物理翻页的延迟，通过键盘硬映射（Ctrl/Alt+字母）解决大脑记忆的易逝性。当您熟练后，翻阅一本厚重的扫描版手册将不再是一场耐心的考验，而是一场指尖与思维的同步舞蹈——左手负责"时空穿梭"（书签与跳转），右手负责"细节审视"（翻页与批注），真正让电子屏幕承载了纸质书独有的空间方位感与信手拈来的从容。
