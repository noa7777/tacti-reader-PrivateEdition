## 🎯 核心设计理念

TactiReader 专为任何高交叉引用密度的长篇文档而生，包括但不限于：  
- 技术标准与协议规范（如 IEEE、ISO）  
- 学术专著与教科书  
- 法律法规与政策文件  
- 工程手册与设计指南  
- 医学指南与药典  

我们坚信：  
- 专业阅读的本质是比对与关联，而非线性滚动。  
- 效率来自肌肉记忆：十指覆盖 Q 到 P，十页瞬达。  
- 系统应自动理解你的意图，而非强迫你手动设置。  

因此，TactiReader 引入了以下核心机制：  
- 智能双页锁定：左窗格固定参考，右窗格自由浏览  
- Home 进度追踪：自动记录你到达过的最远页  
- 瞬时书签（Q-P）：键盘十键盲操设/跳/删书签  
- 目录导航：自动解析 PDF 内嵌目录，一键跳转章节  
- 十键倍数跳转（1-9 + <-/->）：快速跨页导航  
- 全文搜索高亮：F 搜索后，所有匹配项自动高亮  
- 单步导航回退（Ctrl+A）：仅返回上一次跳转前的位置  
- 精准页码跳转（G + Ctrl+G）：支持物理页码校正  
- 焦点窗格机制：所有操作精准作用于当前窗格  

让千页文档如掌上观纹，思维不被打断。

## ⌨️ 精准快捷键指南

### 🔁 基础导航

- `A` / `D`：上一页 / 下一页  
- `<-` / `->`：向后或向前跳转 N 页（N 由 1 到 9 设置）  
- `1`-`9`：设置跳转倍数 N  
- `F`：全文搜索（仅支持含文字的 PDF）  
  - 搜索结果自动高亮，按 Enter 跳转到下一个匹配项  
- `G`：跳转到指定页码（输入数字后回车）  
- `Ctrl + G`：标定物理页码偏移  
  - 在当前页输入其实际逻辑页码（如此处应为第 1 页）  
  - 标定后，G 跳转和书签页码均按校正后的逻辑页计算  
- `Ctrl + Shift + G`：清除物理页码偏移  
- `Ctrl + A`：单步导航回退  
  - 仅返回上一次跨页跳转前的位置  
  - 左右窗格独立记录，仅支持一步回退  

### 🏠 Home：你的阅读前沿

Home 不是锚点，而是你到达过的最远页码。  
当你通过 D、->、G、搜索或书签跳转到一个大于当前 Home 的页面时，Home 自动更新。  
向后翻页（A）不会降低 Home。  

- `空格`：瞬间返回 Home，回到你已读区域的最前沿  
- `Ctrl + 空格`：手动重设 Home 为当前焦点窗格的页面  

### 📌 瞬时书签：十键即达

- `Ctrl + [Q, W, E, R, T, Y, U, I, O, P]`：在当前焦点窗格页面设书签  
  - 可命名，也可留空  
- `[Q-P]`：跳转到对应书签页  
  - 跳转目标始终是当前获得键盘焦点的窗格  
- `Alt + [Q-P]`：清除对应书签  

Q 到 P 是键盘顶行十键，无需移动手指，形成盲操直觉。

### 📚 目录与书签面板

- 如果 PDF 内嵌了目录，TactiReader 会自动解析并显示为可点击的树形结构  
- 点击目录条目可直接跳转到对应章节，作用于当前焦点窗格  
- 点击目录图标可以收起或展开目录
### 🔍 焦点窗格机制

TactiReader 的所有页面操作都作用于当前焦点窗格。

- 点击左窗格 -> 左窗格获得焦点（边框变红）  
- 点击右窗格 -> 右窗格获得焦点  
- 初始状态：右窗格默认获得焦点  
- 所有快捷键操作只影响当前焦点窗格  

在双页模式下，左窗格可被锁定：  
- 一旦你主动操作左窗格（点击后翻页或跳转书签），它将自动锁定  
- 此后，右窗格自由翻阅，左窗格固定为参考页,左窗格也可以独立自由翻页调整参考页,两页之间不互相影响.  

- `C`：切换左窗格的锁定与跟随状态  
- 跟随状态下,左窗格自动跟随为右窗格页的前一页.  
- 锁定状态下,左页不随着右页翻页,但左页可以独立翻页调整参考内容.

### 🔄 视图与布局

- `Z`：切换单页或双页视图  
- `N`：显示或隐藏左侧书签与目录面板  
- `S`：立即保存当前配置（包括批注、书签、旋转、Home、页码偏移等）  
- `F11`：进入或退出全屏模式  

#### 鼠标操作
- 拖拽页面：平移视图  
- Ctrl + 滚轮：缩放页面  
- 拖拽中间分隔条：调整左右窗格宽度比例

### 🧭 页面操作

- `~`（Tab 上方）：顺时针旋转当前焦点窗格 90 度  
- `X`：重置当前焦点窗格（缩放、平移、旋转）  
- `Ctrl + X`：重置双页布局，分隔条居中  
- `Ctrl + Shift + X`：清除全书所有页面的旋转状态  
- `Ctrl + Shift + R`：全局重置  
  - 清除所有书签、批注、Home、旋转、页码偏移等  
  - 恢复到首次打开文档时的原始状态  
  - 此操作不可撤销，请慎用  

### ✏️ 批注系统

- `B`：进入自由画笔模式  
- `H`：进入矩形高亮模式  
- `V`：进入文本批注模式（框随页面旋转，文字始终正立）  
  - 输入后按 Ctrl + Enter 确认（Enter 仅换行）  
- `Esc`：退出当前批注模式或文字选取模式  

#### 批注管理
- `Ctrl + Z`：撤销最后一条批注  
- `Ctrl + Shift + C`：清空当前页所有批注

### 📝 文字交互

- `J`：切换文字选取模式  
  - 进入后，鼠标拖拽可选中文本  
  - 选中后可按 Ctrl+C 复制，或右键点击选择“复制”  
- 退出选取模式：再按 `J` 或按 `Esc`

### 📁 文件与数据机制

#### 文件操作（通过菜单）
- 打开 PDF：加载新文档  
- 另存为 PDF：将当前带批注的页面导出为新 PDF  
- 导出笔记：将当前文档的所有批注、书签、配置打包为 .tactinote 文件  
- 打开笔记：加载 .tactinote 文件（包含 PDF 与全部笔记）  

笔记是一个压缩包，内含原始 PDF 和 session.json（记录所有状态）。  
分享 .tactinote 文件，对方即可获得完全相同的阅读环境。


### 🗂️ 多标签页支持：并行阅读，互不干扰

TactiReader 现已支持同时打开多个 PDF 或 `.tactinote` 笔记本，每个文档独占一个标签页，状态完全隔离。

- 打开新文档时，自动创建新标签页  
- 标签页可拖拽排序，移动到标签右侧会呼出关闭按钮可点击关闭  
- 当前活动标签页的内容即为主窗口显示内容  

#### 右键在新窗口打开
- 在标签页上右键
- 选择【在新窗口中打开】  
- 即可启动一个独立的 TactiReader 实例，实现真正的多文档并行比对  

> 此功能特别适用于：  
> - 同时对照两份不同版本的标准文档  
> - 边看教材边查阅习题解答  
> - 主文档与附录/参考手册分离阅读  

所有标签页共享最近文件列表，但各自保存独立的书签、批注、Home 位置与页码偏移。切换标签即刻还原完整阅读上下文。

#### 数据持久化
所有数据自动保存至：  
`%APPDATA%\TactiReader\tactireader_bookmarks\`  
每个 PDF 有独立配置文件，关闭软件后重新打开，状态完全还原。

## 💡 谢同学的解决方案

1. 打开 `Industrial Protocol Standard.pdf`  
2. 发现前 9 页是封面和目录，正文从第 10 页开始标为 1  
3. 在第 10 页按 Ctrl+G，输入 1 完成物理页码标定  
4. 按 G 输入 87 回车，跳转到真正的第 87 页（帧结构定义）  
5. 按 Ctrl+Q 设书签“帧结构”  
6. 通过目录进入 Chapter 7，按 3 + `->` 定位错误码 0x1A，按 Ctrl+W 设书签  
7. 在 Appendix B 找到通信时序图，按 Ctrl+E 设书签  
8. 在 Chapter 5 找到电气参数表，按 Ctrl+R 设书签  
9. 点击左窗格，按 Q 锁定帧结构；点击右窗格，按 W 查看错误码  
10. 按 F 搜 0x1A，高亮查看；无关内容？Ctrl+A 立刻返回。主线中断？空格回 Home。完成后按 S 保存。
