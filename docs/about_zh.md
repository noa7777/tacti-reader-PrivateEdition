# About TactiReader 1.0

**TactiReader** 是一款轻量级 PDF 阅读器，专为**高交叉引用密度的长篇文档**设计，解决专业阅读中的比对与跳转难题。
> <small>Scroll down for the English version.</small>
## 🧠 核心理念
- **模拟纸质书的空间直觉**
- **键盘驱动，零鼠标依赖**
- **状态持久化，开箱即用**

## 🛠 技术栈
- Python 3.14+
- PyQt5
- PyMuPDF (fitz)
- markdown

## 📦 数据存储
所有书签、批注、视图状态自动保存至：

%APPDATA%\TactiReader\tactireader_bookmarks\

## 👨‍💻 关于我

我是**浙江大学机器人工程专业的大三学生 谢同学**。  
在阅读各类技术文档（从通信协议到数学证明）的过程中，我深感现有 PDF 阅读器对**非线性、关联式阅读**的支持严重不足。

于是，我利用课余时间独立开发了 TactiReader —— **它不关心你读什么，只关心你如何高效地连接知识**。

我相信：**好的工具应该消失在使用中，只留下思考的流畅**。
